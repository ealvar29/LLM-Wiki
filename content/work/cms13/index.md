# Optimizely CMS 13

Optimizely CMS 13 is the latest major version of the Optimizely Content Management System (formerly Episerver). It targets .NET 8+ and introduces significant architectural changes over CMS 12.

## Key Changes from CMS 12

- Built on **.NET 8** (long-term support)
- Improved **content delivery API** for headless scenarios
- Updated **dependency injection** patterns throughout
- New **UI shell** with modernized editing experience
- Better support for **cloud-native** deployments (Azure, DXP)
